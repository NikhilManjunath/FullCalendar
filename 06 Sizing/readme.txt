Sizing is used to control the dimensions of the calendar

Different sizing parameters can be used such as:
height: 'auto',
-> sets the height of the entire calendar, including header and footer
->sticky header dates will be activated

contentHeight: 600
-> Sets the height of the view area of the calendar.

aspectRatio: 2
->Sets the width-to-height aspect ratio of the calendar.

expandRows: true
->If the rows of a given view donâ€™t take up the entire height, they will expand to fit.

handleWindowResize: true
->default true i,e. automatically resize the calendar when the browser window resizes.

windowResizeDelay: 100
->calendar will wait to adjust its size after a window resize occurs, in milliseconds.
