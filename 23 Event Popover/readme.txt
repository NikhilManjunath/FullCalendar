When not all events will fit in a given day, 
you can display the excess events in a small window that will show up when the user clicks a “more” link.

dayMaxEventRows: true,  
->The max number of stacked event levels within a given day.

views: {
  dayGrid: {
    dayMaxEventRows: 4
  }
},
->if more than 4 events, a popup window generated
