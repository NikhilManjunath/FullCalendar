Toolbars are used to defin the buttons at the top and bottom of the calendar

headerToolbar: {
          start: 'today,prev,next',
          center: 'title',
          end: 'dayGridMonth,timeGridWeek,timeGridDay,listWeek'
        },
-> displays these buttons at the top of the toolbar
                
footerToolbar: {
          start: 'prevYear',
          center: '',
          end: 'nextYear'
        },
-> displays these buttons at the top of the toolbar
