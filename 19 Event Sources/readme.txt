Events can be added from 3 sources:
 1. Array: An array of Event Objects that will be displayed on the calendar.

 2. json feed: A URL of a JSON feed that the calendar will fetch Event Objects from.
 events: 'https://fullcalendar.io/demo-events.json?with-resources=4&single-day'

 3. Function: A custom function for programmatically generating Events.

 4. From Google Calendar: FullCalendar can display events from a public Google Calendar.
  events: 'en.usa#holiday@group.v.calendar.google.com'
